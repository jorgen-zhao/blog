## Material Design 3 核心概念

在开始编码之前，让我们先了解一下Material Design 3的几个核心概念：

### 1. Material You - 个性化设计

Material You是Material Design 3的核心理念，它允许系统根据用户的壁纸自动提取颜色，生成个性化的主题。这意味着每个用户的设备都会有独特的视觉体验！

### 2. 动态颜色

动态颜色系统基于Material You概念，可以根据系统生成的颜色方案自动调整UI元素的颜色，确保整个应用视觉的一致性。

### 3. 形态与层次

Material Design 3引入了更丰富的形态和层次感，通过新的阴影系统、边界和表面，创造出更加立体的视觉体验。

### 4. 运动与动画

新的设计语言更注重微妙的动画效果，使界面交互更加流畅自然，提升用户体验。

## 实现Material Design 3

现在，让我们来看看如何在Android应用中实现Material Design 3！

### 添加依赖

首先，确保你的项目中添加了Material Components for Android的依赖：

```gradle
dependencies {
    implementation 'com.google.android.material:material:1.9.0'
}
```

### 配置主题

在`res/values/themes.xml`中配置你的应用主题：

```xml
<resources xmlns:tools="http://schemas.android.com/tools">
    <!-- Base application theme. -->
    <style name="Theme.MyApp" parent="Theme.Material3.DayNight.NoActionBar">
        <!-- Primary brand color. -->
        <item name="colorPrimary">@color/purple_500</item>
        <item name="colorPrimaryVariant">@color/purple_700</item>
        <item name="colorOnPrimary">@color/white</item>
        <!-- Secondary brand color. -->
        <item name="colorSecondary">@color/teal_200</item>
        <item name="colorSecondaryVariant">@color/teal_700</item>
        <item name="colorOnSecondary">@color/black</item>
        <!-- Status bar color. -->
        <item name="android:statusBarColor">?attr/colorPrimaryVariant</item>
    </style>
</resources>
```

### 启用动态颜色

要启用动态颜色功能，需要在主题中添加`android:dynamicColor`属性：

```xml
<style name="Theme.MyApp" parent="Theme.Material3.DayNight.NoActionBar">
    ...
    <item name="android:dynamicColor">true</item>
</style>
```

### 使用Material组件

Material Design 3提供了丰富的UI组件，让我们来看看如何在Jetpack Compose中使用它们：

```kotlin
import androidx.compose.material3.*
import androidx.compose.runtime.*

@Composable
fun Material3Example() {
    var selected by remember { mutableStateOf(false) }
    
    Column(
        modifier = Modifier
            .fillMaxSize()
            .padding(16.dp),
        horizontalAlignment = Alignment.CenterHorizontally,
        verticalArrangement = Arrangement.spacedBy(16.dp)
    ) {
        // Material 3 Button
        Button(onClick = { selected = !selected }) {
            Text(text = if (selected) "已选择" else "选择我")
        }
        
        // Material 3 Elevated Button
        ElevatedButton(onClick = { /* do something */ }) {
            Text(text = "提升按钮")
        }
        
        // Material 3 Outlined Button
        OutlinedButton(onClick = { /* do something */ }) {
            Text(text = "轮廓按钮")
        }
        
        // Material 3 Card
        Card(
            modifier = Modifier.fillMaxWidth(),
            elevation = CardDefaults.cardElevation(defaultElevation = 8.dp)
        ) {
            Column(
                modifier = Modifier.padding(16.dp)
            ) {
                Text(
                    text = "卡片标题",
                    style = MaterialTheme.typography.headlineSmall
                )
                Text(
                    text = "这是卡片内容，展示Material Design 3的样式效果",
                    style = MaterialTheme.typography.bodyMedium
                )
            }
        }
        
        // Material 3 Switch
        Row(
            verticalAlignment = Alignment.CenterVertically,
            modifier = Modifier.fillMaxWidth()
        ) {
            Text(text = "切换开关")
            Switch(
                checked = selected,
                onCheckedChange = { selected = it },
                modifier = Modifier.padding(start = 8.dp)
            )
        }
    }
}
```

### 自定义颜色方案

如果你想创建自己的颜色方案，而不依赖于系统生成的动态颜色，可以这样做：

```kotlin
@Composable
fun MyAppTheme(
    darkTheme: Boolean = isSystemInDarkTheme(),
    content: @Composable () -> Unit
) {
    val colorScheme = if (darkTheme) {
        darkColorScheme(
            primary = Color(0xFFBB86FC),
            secondary = Color(0xFF03DAC5),
            tertiary = Color(0xFFCF6679),
            background = Color(0xFF121212),
            surface = Color(0xFF121212),
            onPrimary = Color.Black,
            onSecondary = Color.Black,
            onTertiary = Color.Black,
            onBackground = Color.White,
            onSurface = Color.White,
        )
    } else {
        lightColorScheme(
            primary = Color(0xFF6200EE),
            secondary = Color(0xFF03DAC5),
            tertiary = Color(0xFFCF6679),
            background = Color.White,
            surface = Color.White,
            onPrimary = Color.White,
            onSecondary = Color.Black,
            onTertiary = Color.Black,
            onBackground = Color.Black,
            onSurface = Color.Black,
        )
    }

    MaterialTheme(
        colorScheme = colorScheme,
        typography = Typography(),
        shapes = MaterialTheme.shapes,
        content = content
    )
}
```

## Material Design 3 最佳实践

### 1. 颜色使用原则

- **主色调**：用于主要操作、交互元素和品牌标识
- **次色调**：用于次要操作和强调元素
- **表面颜色**：用于卡片、表单等表面元素
- **文本颜色**：确保足够的对比度，保证可读性

::: theorem
颜色对比度原则：文本与背景的对比度至少应为4.5:1，以确保良好的可读性。
:::

### 2. 字体与排版

Material Design 3提供了改进的排版系统，包括更清晰的层次结构和更好的阅读体验：

## 实战案例：Material Design 3应用界面

让我们通过一个简单的示例，展示如何使用Material Design 3构建一个现代应用界面：

```kotlin
@Composable
fun Material3AppScreen() {
    var selectedItem by remember { mutableStateOf(0) }
    
    Scaffold(
        containerColor = MaterialTheme.colorScheme.background,
        topBar = {
            TopAppBar(
                title = {
                    Text("Material Design 3 示例")
                },
                colors = TopAppBarDefaults.topAppBarColors(
                    containerColor = MaterialTheme.colorScheme.primary,
                    titleContentColor = MaterialTheme.colorScheme.onPrimary
                )
            )
        },
        bottomBar = {
            NavigationBar(
                containerColor = MaterialTheme.colorScheme.surface,
                contentColor = MaterialTheme.colorScheme.onSurface
            ) {
                listOf("首页", "发现", "消息", "我的").forEachIndexed { index, title ->
                    NavigationBarItem(
                        icon = {
                            Icon(
                                imageVector = when (index) {
                                    0 -> Icons.Default.Home
                                    1 -> Icons.Default.Explore
                                    2 -> Icons.Default.Notifications
                                    else -> Icons.Default.Person
                                },
                                contentDescription = title
                            )
                        },
                        label = { Text(title) },
                        selected = selectedItem == index,
                        onClick = { selectedItem = index },
                        colors = NavigationBarItemDefaults.colors(
                            selectedIconColor = MaterialTheme.colorScheme.primary,
                            selectedTextColor = MaterialTheme.colorScheme.primary,
                            unselectedIconColor = MaterialTheme.colorScheme.onSurface,
                            unselectedTextColor = MaterialTheme.colorScheme.onSurface,
                            indicatorColor = MaterialTheme.colorScheme.primaryContainer
                        )
                    )
                }
            }
        }
    ) { paddingValues ->
        Column(
            modifier = Modifier
                .fillMaxSize()
                .padding(paddingValues)
                .padding(16.dp),
            horizontalAlignment = Alignment.CenterHorizontally,
            verticalArrangement = Arrangement.spacedBy(16.dp)
        ) {
            Card(
                modifier = Modifier.fillMaxWidth(),
                colors = CardDefaults.cardColors(
                    containerColor = MaterialTheme.colorScheme.surface
                ),
                elevation = CardDefaults.cardElevation(defaultElevation = 4.dp)
            ) {
                Column(
                    modifier = Modifier.padding(16.dp)
                ) {
                    Text(
                        text = "欢迎来到Material Design 3",
                        style = MaterialTheme.typography.headlineSmall,
                        color = MaterialTheme.colorScheme.onSurface
                    )
                    Spacer(modifier = Modifier.height(8.dp))
                    Text(
                        text = "这是一个展示Material Design 3特性的示例界面",
                        style = MaterialTheme.typography.bodyMedium,
                        color = MaterialTheme.colorScheme.onSurface
                    )
                }
            }
            
            Row(
                modifier = Modifier.fillMaxWidth(),
                horizontalArrangement = Arrangement.spacedBy(16.dp)
            ) {
                FilledIconButton(
                    onClick = { /* do something */ },
                    colors = ButtonDefaults.filledIconButtonColors(
                        containerColor = MaterialTheme.colorScheme.primary,
                        contentColor = MaterialTheme.colorScheme.onPrimary
                    )
                ) {
                    Icon(Icons.Default.Favorite, contentDescription = "喜欢")
                }
                
                OutlinedIconButton(
                    onClick = { /* do something */ },
                    colors = ButtonDefaults.outlinedIconButtonColors(
                        contentColor = MaterialTheme.colorScheme.primary
                    )
                ) {
                    Icon(Icons.Default.Share, contentDescription = "分享")
                }
            }
            
            ProgressIndicator()
        }
    }
}

@Composable
fun ProgressIndicator() {
    var progress by remember { mutableFloatStateOf(0.1f) }
    var loading by remember { mutableStateOf(false) }
    
    Column(
        verticalArrangement = Arrangement.spacedBy(8.dp),
        horizontalAlignment = Alignment.CenterHorizontally
    ) {
        LinearProgressIndicator(
            progress = progress,
            modifier = Modifier.fillMaxWidth(),
            color = MaterialTheme.colorScheme.primary,
            trackColor = MaterialTheme.colorScheme.primaryContainer
        )
        
        Text(
            text = "${(progress * 100).toInt()}%",
            style = MaterialTheme.typography.bodySmall
        )
        
        Button(onClick = {
            loading = true
            // 模拟加载过程
            kotlinx.coroutines.GlobalScope.launch {
                repeat(10) {
                    delay(300)
                    progress += 0.1f
                }
                loading = false
            }
        }) {
            if (loading) {
                CircularProgressIndicator(
                    modifier = Modifier.size(16.dp),
                    color = MaterialTheme.colorScheme.onPrimary,
                    strokeWidth = 2.dp
                )
                Spacer(modifier = Modifier.width(8.dp))
            }
            Text(text = if (loading) "加载中..." else "开始加载")
        }
    }
}
```
