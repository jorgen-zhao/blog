## Docker Volumes

**When do we need Docker Volumes?**

Data is gone! When restarting or removing the container. The container has virtual file system which is mapping the hostâ€™s physical file system.

**3 Volume Types**

1. Host Volumeï¼šyou decide **where on the host file system** the reference is made.

```bash
docker run -v host file dir : container file dir
```

2. Anonymous Volumesï¼šfor each container a folder is generated that gets mounted.

```bash
docker run -v /var/lib/mysql/data --> /var/lib/docker/volumes/random-hash/_data
```

3. Named Volumesï¼šyou can **reference** the volume by **name**; ï¼ˆğŸ‘ğŸ‘ğŸ‘ï¼‰

```bash
docker run -v name:/var/lib/mysql/data
```
