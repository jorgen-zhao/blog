## 前言

这个元旦假期，我决定放下手机，深度体验一下当前的AI浪潮。国内有智谱和豆包，国外有Google，这些工具简直像打开了新世界的大门。🤯 我试玩了各种功能，从开发实用app到自动化博客写作，再到畅想未来创作，整个过程让我感慨万千。AI的发展速度太快了，2025年就像坐上了火箭，🚀 但说实话，LLM也让我更忙了——每天事情多到爆炸，却又充实得不行。今天就来聊聊我的AI之旅，顺便分享一些技术心得和生活感悟。

## AI拍照app的开发：智能姿势指导

这个假期，我突发奇想，用Google的antigravity平台编写了一个小app，专门帮男性朋友给女朋友拍照。~~别笑，这可是硬核技术活~~！app的核心逻辑很简单：用户上传照片后，调用智谱的场景识别功能，分析背景环境（比如公园、咖啡馆），然后根据场景推荐最佳拍照姿势。最后，用豆包生成姿势图片作为参考。整个过程超级丝滑，快到飞起——只要描述到位，基本就能完美实现意图。

- **技术栈**：Google antigravity（开发框架） + 智谱AI（场景识别） + 豆包（姿势生成）。
- **关键点**：智谱的识别速度超快，准确率高达90%以上；豆包的生成图片质量也很高，细节丰富。
- **个人吐槽**：测试时，朋友们的反馈是“太实用了”，但我自己试了试，结果姿势僵硬得像机器人，🤷‍♂️ 看来AI再智能，也得靠人来演绎啊！

::: theorem  
**AI辅助创作的优势**：效率提升 + 个性化输出。但别忘了，人类创意才是灵魂！  
:::

## 博客自动化之旅：AI改写 + GitHub Actions

除了拍照app，我还折腾了博客写作的自动化。以前写博客，要么是手动敲字，要么是记录碎片化随笔，效率低下。现在，我引入了AI来帮忙：

1. **深度报告生成**：用Google的deep research功能形成初步报告，然后让AI根据我的行文风格改写，确保文章“有Jorgen味儿”。
2. **随笔扩写**：把日常的思维碎片（比如今天的一些想法）丢给AI，让它扩写成完整段落。
3. **自动化流程**：使用智谱AI搭配GitHub Actions，通过创建issue触发文章编/改写。完成后，自动提交代码、pull request，再结合CI（持续集成）发布到博客网站。整个过程非常丝滑，省时省力！

::: right  
**技术亮点**：GitHub Actions + 智谱API + CI/CD，实现“零手动”博客发布。  
:::

- **为什么高效**：AI能直接理解意图，描述到位就能生成高质量内容。比如，我输入“元旦AI体验”，它就能扩展成几百字的技术感悟。
- **幽默时刻**：有次AI改写我的随笔，结果加了太多“哈哈哈”，搞得文章像段子集，😂 我赶紧手动调整——看来AI也需要“调教”啊！

## AI发展的感慨：起飞的2025与忙碌的2026

真的，2025年AI的发展就像起飞了一样。从简单的聊天机器人到现在的多模态AI（文字、图像、视频），变化太快了。我亲身体验到，LLM（大语言模型）不仅涉足了音乐、视频领域，还渗透到生活方方面面。比如，我用豆包生成姿势图片时，它还能同步输出音乐建议，🎵 这太神奇了。

但另一方面，AI也让我更累了。怎么说呢？因为LLM能直接理解并且实现意图，每天可干的事情就多了——写报告、改文章、开发app，甚至规划未来。现在，我的待办清单上堆满了“用LLM创作科幻短剧”的计划，哈哈哈。~~每天都是忙碌且充实的一天，但头发也掉得更快了~~。

> **核心结论**：AI是双刃剑，它解放了生产力，却也增加了认知负担。关键在于，我们要学会与AI协作，而不是被它奴役。

## 未来展望：LLM创作科幻短剧

展望2026年，我计划用LLM来创作科幻短剧。想象一下：输入一个简单的故事框架，AI就能生成剧本、分镜、甚至音乐。这不再是科幻电影里的场景，而是触手可及的现实。~~到时候，说不定我还能靠这个赚点零花钱呢~~！

- **初步想法**：结合智谱和豆包，构建一个创作流水线。GitHub Actions自动触发AI生成，然后发布到博客或视频平台。
- **期待**：2026年AI会更强大吗？或许会出现“通用人工智能”的雏形。但不管怎样，保持好奇心和创造力，才是王道。

::: tip  
**给读者的建议**：拥抱AI，但别迷失在技术中。多花时间体验生活，让AI成为你的助手，而不是替代品。  
:::
