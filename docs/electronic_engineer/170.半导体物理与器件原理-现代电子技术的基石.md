## 半导体的基本概念

半导体，顾名思义，是导电性介于导体和绝缘体之间的材料。在电子工程的世界里，它们就像是"魔法材料"，通过掺杂和外部激励，可以改变其导电特性。

### 本征半导体

纯半导体材料（如硅、锗）被称为本征半导体。在这些材料中，电子和空穴成对出现，形成本征载流子。

- 硅(Si)和锗(Ge)是最常用的本征半导体材料
- 在绝对零度时，本征半导体表现为绝缘体
- 随着温度升高，载流子浓度增加

```python
# 简化的本征载流子浓度计算
import numpy as np

def intrinsic_carrier_concentration(T, Eg=1.12, Nc=2.8e19, Nv=1.04e19):
    """
    计算硅的本征载流子浓度
    T: 温度(K)
    Eg: 禁带宽度(eV)，默认为硅的1.12eV
    Nc: 导带有效态密度(cm^-3)
    Nv: 价带有效态密度(cm^-3)
    """
    k = 8.617e-5  # 玻尔兹曼常数(eV/K)
    ni = np.sqrt(Nc * Nv) * np.exp(-Eg / (2 * k * T))
    return ni

# 在300K(室温)下计算
print(f"300K时硅的本征载流子浓度: {intrinsic_carrier_concentration(300):.2e} cm^-3")
```

### 掺杂半导体

通过向本征半导体中掺入少量杂质，可以显著改变其导电特性：

- **N型半导体**：掺入五价元素（如磷、砷），提供多余电子
- **P型半导体**：掺入三价元素（如硼、铟），提供空穴

::: theorem
PN结是半导体器件的基本构建块，通过P型半导体和N型半导体的结合形成。
:::

## 核心半导体器件

### 二极管

二极管是最简单的半导体器件之一，具有单向导电性。

**工作原理**：
- 正向偏置：P区接正极，N区接负极，电流容易通过
- 反向偏置：P区接负极，N区接正极，电流几乎不通过

```
    正向偏置                  反向偏置
      +                         -
      |                         |
     / \                       / \
    /   \                     /   \
   P     N                   P     N
    \   /                     \   /
     \ /                       \ /
      |                         |
      -                         +
```

### 双极结型晶体管(BJT)

BJT是现代电子学中最重要的器件之一，具有电流放大作用。

**结构**：
- NPN型：N型发射极、P型基极、N型集电极
- PNP型：P型发射极、N型基极、P型集电极

**工作原理**：
- 基极电流控制集电极电流
- 具有电流放大作用

### 场效应晶体管(FET)

FET是另一种重要的晶体管类型，具有高输入阻抗和低功耗特性。

**主要类型**：
- 结型场效应晶体管(JFET)
- 金属氧化物半导体场效应晶体管(MOSFET)

MOSFET在现代集成电路中占据主导地位，是CPU、内存等芯片的基本构建单元。

## 现代半导体技术发展

### 尺寸微缩与摩尔定律

半导体器件的尺寸不断缩小，遵循摩尔定律：集成电路上可容纳的元器件数目约每隔18-24个月便会增加一倍。

```
年份    | 特征尺寸(nm)
--------|------------
2010    | 45
2013    | 22
2016    | 14
2019    | 7
2022    | 5
2025    | 3
```

### 新型半导体材料

除了传统的硅材料，新型半导体材料正在崛起：

- **碳纳米管**：具有优异的电学性能
- **石墨烯**：超高电子迁移率
- **氮化镓(GaN)**：高功率、高频率应用
- **碳化硅(SiC)**：高温、高压应用

## 半导体器件的应用

### 逻辑电路

半导体器件是数字逻辑电路的基础，构成了计算机、智能手机等现代电子设备的核心。

### 电源管理

功率半导体器件（如MOSFET、IGBT）在电源转换、电机控制等领域发挥关键作用。

### 射频应用

射频半导体器件用于无线通信、雷达等高频应用。

### 传感器

半导体传感器将物理量（如温度、压力、光）转换为电信号。
