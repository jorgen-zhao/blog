## 电路分析的核心概念

### 1. 基本电路元件

电路就像一个微型城市，各种元件扮演着不同角色：

| 元件 | 符号 | 功能 | 常见应用 |
|------|------|------|----------|
| 电阻 | 🟫 | 限制电流 | 分压、限流 |
| 电容 | 🟦 | 储存电荷 | 滤波、定时 |
| 电感 | 🟪 | 储存磁能 | 振荡、滤波 |
| 电源 | 🔋 | 提供能量 | 所有电路 |

> 💡 **小贴士**：在面包板上搭建电路时，记得先检查电源极性！😅

### 2. 欧姆定律

::: theorem
欧姆定律是电路分析的基石：**V = I × R**
:::

其中：
- V：电压（伏特）
- I：电流（安培）
- R：电阻（欧姆）

**实际应用**：
```
假设一个5V电源连接100Ω电阻：
I = V/R = 5V/100Ω = 50mA
```

### 3. 基尔霍夫定律

::: tip
基尔霍夫定律是分析复杂电路的"万能钥匙"：
1. **电流定律(KCL)**：节点处电流代数和为零
2. **电压定律(KVL)**：闭合回路电压代数和为零
:::

**KCL示例**：
```
在节点A：
I1 + I2 = I3
```

## 实践技巧

### 1. 分压电路设计

当需要从9V电源获取5V电压时：

```
R1 = 4kΩ
R2 = 5kΩ
Vout = Vin × (R2/(R1+R2)) = 9V × (5k/9k) = 5V
```

### 2. 滤波电路

电容滤波电路可减少纹波：

```
┌───┐     ┌───┐
│   ├─┬───┤   │
│   │ │   │   │
└───┘ └───┘   │
              │
```
